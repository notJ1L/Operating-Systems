Windows 10 Installation

Follow along with this comprehensive guide to install Windows 10 on your system.

Video Tutorial

Installation Steps

  1. Create a bootable USB drive using Microsoft's Media Creation Tool or Rufus.
  2. Enter your computer's BIOS/UEFI and set the USB drive as the primary boot device.
  3. Restart your computer and follow the Windows setup wizard prompts.
  4. Select your language, time zone, and keyboard layout.
  5. Click "Install now" and enter your product key (or choose "I don't have a product key").
  6. Select the edition of Windows 10 you want to install.
  7. Accept the license terms and choose "Custom: Install Windows only (advanced)".
  8. Select the drive/partition for installation and proceed with the setup.
  9. Wait for the installation to complete; your computer will restart several times.
  10. Complete the out-of-box experience (OOBE) setup for your account and preferences.

DHCP Roles and Domain

This activity covers configuring DHCP (Dynamic Host Configuration Protocol) roles within a Windows Server 2008 environment and integrating with Active Directory Domain Services. You will learn how to install the DHCP server role, authorize the server in the domain, create scopes, and configure options for automatic IP address allocation to clients on your network.

Video Tutorial

Step-by-Step Process

  1. Open Server Manager and add the DHCP Server role from the Add Roles Wizard.
  2. Authorize the DHCP server in Active Directory (if joined to a domain).
  3. Create a new IPv4 scope with the desired IP range and subnet mask.
  4. Configure scope options (default gateway, DNS servers, WINS if needed).
  5. Activate the scope and verify DHCP server operation.
  6. Test from a client by setting network adapter to obtain IP automatically.
